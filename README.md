# Rakin Talukder - Portfolio

A highly engineered, responsive personal portfolio built with Flutter Web. 

## Design Language: Neo-Brutalism & Minimalism
The design aesthetic of this portfolio is best described as a refined **Neo-Brutalism** mixed with **Minimalism**. 

Key characteristics of this theme include:
- **Bold Typography:** Heavy reliance on stark, highly legible geometric fonts (Space Grotesk) to drive the visual hierarchy rather than relying on images or busy layouts.
- **High Contrast:** A stark, uncompromising color palette (pure blacks, stark whites, and ghost greys) that seamlessly adapts to Dark and Light modes.
- **Raw Geometry:** Sharp, deliberate geometric UI components—like the mathematical Radial Edge Menu on mobile and the minimal dot-navigation on desktop.
- **Utilitarian Elegance:** No unnecessary fluff or heavy gradients. Every visual element serves a direct functional purpose, resulting in a highly engineered, premium feel.

## Tech Stack
- **Framework:** Flutter (Web)
- **PDF Generation:** Dart `pdf` and `printing` packages for dynamic resume generation.
- **Responsive Navigation:** Custom Desktop SideNav & Mobile Radial Edge Dial.
- **Animations:** Custom visibility detectors, stagger animations, and animated layouts.

## Running Locally

To run this project on your local machine:

```bash
# Fetch dependencies
flutter pub get

# Run on Chrome
flutter run -d chrome
```
